# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| `main` branch | ✅ |
| Latest release (1.1) | ✅ |
| Older versions | ❌ |
| Alpha builds | ❌ |

## Reporting a Vulnerability

If you have a vulnerability to report, contact me on a [messaging app](https://asterisk.lol/socials) or by email at me@asterisk.lol. I will respond whenever I have a chance.
